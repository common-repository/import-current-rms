Import from Current RMS(Rental Management System) to Woocommerce products using API.

== Description ==

Current RMS plugin fetch products from Current Rental Managment System CRM and import in Woocommerce products along categories



== Installation ==

1. Upload the entire `d_crms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Current RMS' menu in your WordPress admin panel.
